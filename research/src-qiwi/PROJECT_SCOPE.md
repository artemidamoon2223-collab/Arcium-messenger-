# Project Scope — SRC/QIWI

SRC/QIWI is an isolated research memory area within the Arcium
Messenger repository (`research/src-qiwi/`). It exists to preserve an
experimental research agent's reasoning state — observations,
predictions, cases, and handover notes — as durable files instead of
relying on chat history that does not survive `/clear`.

## What this is

- A place to record sourced observations and track claim status over
  time.
- A handover point so a future agent can continue from repository
  artifacts instead of inherited chat context.

## What this is not

- Not messenger application code.
- Not a new agent, service, or runtime component.
- Not a package or dependency.
